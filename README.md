# ECC3479 Research Project — Vacancy Rates and Rent Growth

**Author:** Anton Kozlovsky (36194239)
**Course:** ECC3479 Data and Evidence in Economics, Monash University

## Research Question

> What is the effect of a suburb's rental vacancy rate on subsequent rent
> growth, compared with similar suburbs with higher vacancy?

This project uses a suburb-quarter panel for Sydney and Melbourne
(approximately 2018–2025) to estimate the relationship between rental
vacancy rates and future rent changes, using panel fixed effects.

## How to Run This Project from Scratch

### Step 0: Prerequisites

- **Python 3.9+** installed on your machine
- **pip** (comes with Python)
- **Git** (to clone the repository)

### Step 1: Clone the repository

```bash
git clone https://github.com/Anton1321/ECC3479-project-Anton.git
cd ECC3479-project-Anton
```

### Step 2: Install required packages

```bash
pip install -r requirements.txt
```

### Step 3: Obtain the raw data (manual step)

The raw data comes from proprietary sources and cannot be included in
the repository. Follow the detailed instructions in
[`data/raw/README.md`](data/raw/README.md) to download:

1. **Vacancy rate data** from SQM Research → save as `data/raw/vacancy_rates.csv`
2. **Median asking rent data** from SQM Research / Domain → save as `data/raw/median_rents.csv`

See `data/raw/README.md` for the exact column formats expected.

### Step 4: Run the cleaning script

```bash
python code/01_clean_data.py
```

This reads the two raw CSVs, merges them into a suburb-quarter panel,
calculates rent growth and lagged vacancy rates, and saves the result to
`data/clean/suburb_quarter_panel.csv`.

See [`data/clean/CODEBOOK.md`](data/clean/CODEBOOK.md) for a description
of every variable in the output.
